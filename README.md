# IPL Data Analysis
Exploratory analysis of IPL cricket data spanning 16 seasons (2008–2024), 
covering 1,095 matches and 260,920 ball-by-ball deliveries.

## Dataset
matches.csv — 1,095 IPL matches with team, toss, winner and venue details
deliveries.csv — 260,920 deliveries with batter, bowler and run details

## Tools Used
Python, Pandas, NumPy, Matplotlib

## Key Findings
Mumbai Indians lead all-time wins (144), followed by CSK (138)
CSK has the highest win percentage among established teams at 58.2%
V Kohli is the top run scorer — 8,014 runs at a strike rate of 128.5
AD Russell has the highest strike rate among consistent batters — 164.2
SP Narine took 200 wickets with an economy of 6.76
